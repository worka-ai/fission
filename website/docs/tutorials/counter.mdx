# Counter tutorial

This tutorial extends the first app and makes the loop explicit:

- wire the action contract,
- map both controls to reducer updates,
- verify deterministic transitions,
- add one test path before adding more logic.

```rust
#[derive(Action, serde::Serialize, serde::Deserialize, Debug, Clone, PartialEq, Eq)]
struct Increment;

#[derive(Action, serde::Serialize, serde::Deserialize, Debug, Clone, PartialEq, Eq)]
struct Decrement;
```

Keep actions meaningful and small. Build confidence one behavior at a time, then move on to list-style state.
