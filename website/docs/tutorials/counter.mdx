# Counter tutorial

This tutorial extends the first app by reinforcing the typed action loop.

## Learn by changing one flow

- add increment action,
- add decrement action,
- keep both reducers pure,
- verify both transitions in code and behavior.

```rust
#[derive(Action, Serialize, Deserialize, Debug, Clone, PartialEq, Eq)]
struct Increment;

#[derive(Action, Serialize, Deserialize, Debug, Clone, PartialEq, Eq)]
struct Decrement;
```

Then bind them to controls and verify state changes from interaction. The goal is confidence in a deterministic pair of action paths.
