# Platform and deployment

Start with the architecture, then add target-specific output.

## Recommended sequence

1. build and stabilize core behaviors,
2. validate desktop target,
3. validate web/WASM output,
4. add mobile/packaging flow,
5. lock environment settings outside business logic.

## Deployment guidance

Keep target flags in configuration files and project descriptors. This avoids leaking platform-specific behavior into shared widget code.
