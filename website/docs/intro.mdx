# Start here

If this is your first time with Fission, use this short path and do not jump ahead.

Begin by understanding what problem Fission solves, then install, then build a tiny app. This is the fastest way to see the same ideas repeatedly instead of learning isolated examples.

**Read in this order**

1. [What is Fission](/docs/getting-started/what-is-fission)
2. [Install and setup](/docs/getting-started/install)
3. [Build your first app](/docs/getting-started/first-app)
4. [Playground-first workflow](/docs/guide/playground-driven-workflow)
5. [Counter tutorial](/docs/tutorials/counter)
6. [Todo tutorial](/docs/tutorials/todo)

Every page answers three questions: what this is, why it is useful, and what to do next. Keep the order above, and by the time you open reference pages you will have already met each concept.

When you are ready for quick lookup, open [Reference overview](/reference/overview).
