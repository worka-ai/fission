# Install and setup

This page shows the minimum commands to get a local Fission project running.

## 1. Add the core crate

Use your existing Rust project setup:

```bash
cargo add fission
```

Optional dependencies may be required by examples (for example error handling helpers).

## 2. Install the CLI

The CLI is required for standard project workflow and target operations.

```bash
cargo install fission-cli
# or, while working against the repo:
cargo install --locked --path crates/cli
```

## 3. Create a project

Scaffold a new app with a single command sequence:

```bash
fission init my-fission-app
cd my-fission-app
```

The generated project includes defaults for workspace layout and target metadata.

## 4. Add targets

Desktop and web are the fastest entry points for feedback. Add them explicitly so behavior can be validated in both environments.

```bash
fission add-target desktop
fission add-target web
```

## 5. Run

Use desktop first for interactive speed, then web for layout and browser behavior.

```bash
fission run desktop
fission run web
```
