# Charts reference

Fission Charts gives application code typed Rust chart structures that lower into the same rendering pipeline as the rest of the UI. Use this reference when you already know the chart family you want and need the exact page for a chart variant, its data shape, interaction notes, and example code.

For a guided introduction, start with the [charts overview](/docs/charts/overview). For data modelling guidance, read [Series & DataSets](/docs/charts/series-and-datasets).
