# Media widgets

Use media widgets intentionally with constraints. They may impact layout, loading behavior, and startup cost.

Core entries: `Image`, `Video`, `Icon`, `Code`, and `Kbd`. Build custom drawing with
`canvas(width, height, painter)` and validate clipping, scaling, and fallback behavior early.

For each media area decide:

- loading strategy,
- fallback behavior,
- accessibility text,
- container constraints and clipping.
