# Row

A layout and composition primitive for predictable structure and spacing.


## What this is for
`Row` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::Row;

let node = Row {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<NodeId>`
- `children`: `Vec<Node>`
- `semantics`: `Option<Semantics>`
- `flex_grow`: `f32`
- `flex_shrink`: `f32`
- `gap`: `Option<f32>`
- `wrap`: `FlexWrap`
- `align_items`: `AlignItems`
- `justify_content`: `JustifyContent`
