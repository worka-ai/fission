# CanvasLowerer

A low-level composition utility for precise layout and rendering placement.


## What this is for
Internal lowerer for the [`canvas()`] free function. Wraps a painter closure that produces child node IDs within a `Group` node, placed inside a fixed-size `Box` layout node.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::CanvasLowerer;

let node = CanvasLowerer {
    ..Default::default()
}.into_node();
```

## Config surface
- `width`: `Option<f32>`
- `height`: `Option<f32>`
- `painter`: `Arc<dyn Fn(&mut LoweringContext) -> Vec<NodeId> + Send + Sync>`
