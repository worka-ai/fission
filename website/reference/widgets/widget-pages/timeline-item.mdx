# TimelineItem

A product-surface widget for status, content grouping, and communication.


## What this is for
`TimelineItem` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::TimelineItem;

let node = TimelineItem {
    ..Default::default()
}.into_node();
```

## Config surface
- `title`: `String`
- `description`: `Option<String>`
- `timestamp`: `Option<String>`
