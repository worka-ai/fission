# FormControl

A utility or bridge type used while composing higher-level widget hierarchies.


## What this is for
`FormControl` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::FormControl;

let node = FormControl {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<WidgetNodeId>`
- `label`: `Option<String>`
- `child`: `Box<Node>`
- `error`: `Option<String>`
- `helper`: `Option<String>`
- `required`: `bool`
