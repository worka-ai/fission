# SelectItem

An input-focused control for converting user intent into typed actions.


## What this is for
`SelectItem` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::SelectItem;

let node = SelectItem {
    ..Default::default()
}.into_node();
```

## Config surface
- `label`: `String`
- `icon`: `Option<String>`
- `on_select`: `ActionEnvelope`
