# canvas

A low-level composition utility for precise layout and rendering placement.


## What this is for
Creates a custom paint node from a closure. The `painter` closure receives a [`LoweringContext`] and returns a list of child node IDs. These are grouped inside a fixed-size box with the given `width` and `height` (both optional).

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::canvas;
let node = canvas(Some(200.0), Some(100.0), |cx| {
    Vec::new()
});
```
