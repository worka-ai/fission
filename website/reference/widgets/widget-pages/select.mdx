# Select

An input-focused control for converting user intent into typed actions.


## What this is for
`Select` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::Select;

let node = Select {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `WidgetNodeId`
- `selected_label`: `Option<String>`
- `items`: `Vec<SelectItem>`
- `is_open`: `bool`
- `on_toggle`: `Option<ActionEnvelope>`
- `placeholder`: `String`
- `width`: `Option<f32>`
