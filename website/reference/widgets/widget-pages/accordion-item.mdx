# AccordionItem

A public widget/API entrypoint in the Fission surface.


## What this is for
`AccordionItem` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::AccordionItem;

let node = AccordionItem {
    ..Default::default()
}.into_node();
```

## Config surface
- `title`: `String`
- `content`: `Node`
- `is_expanded`: `bool`
- `on_toggle`: `Option<ActionEnvelope>`
