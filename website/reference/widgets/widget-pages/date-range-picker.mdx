# DateRangePicker

An input-focused control for converting user intent into typed actions.


## What this is for
`DateRangePicker` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::DateRangePicker;

let node = DateRangePicker {
    ..Default::default()
}.into_node();
```

## Config surface
- `id_start`: `WidgetNodeId`
- `id_end`: `WidgetNodeId`
- `start`: `Option<NaiveDate>`
- `end`: `Option<NaiveDate>`
- `is_start_open`: `bool`
- `is_end_open`: `bool`
- `on_toggle_start`: `Option<ActionEnvelope>`
- `on_toggle_end`: `Option<ActionEnvelope>`
- `on_close_start`: `Option<ActionEnvelope>`
- `on_close_end`: `Option<ActionEnvelope>`
