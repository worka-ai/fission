# CustomNode

A utility or bridge type used while composing higher-level widget hierarchies.


## What this is for
`CustomNode` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::CustomNode;

let node = CustomNode {
    ..Default::default()
}.into_node();
```

## Config surface
- `debug_tag`: `String`
- `lowerer`: `Option<Arc<dyn LowerDyn>>`
- `render_object`: `Option<Arc<dyn CustomRenderObject>>`
