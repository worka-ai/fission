# flyout

A low-level composition utility for precise layout and rendering placement.


## What this is for
Positions `content` relative to an `anchor` node using the flyout layout system. The layout engine places the content adjacent to the anchor's computed rect. This is the foundation for [`Popover`], [`Tooltip`], [`Menu`], and [`Select`] popups.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::NodeId;
use fission_widgets::flyout;
let flyout_node = flyout(NodeId::from_u64(0), fission_core::ui::Text::new("menu").into_node());
```
