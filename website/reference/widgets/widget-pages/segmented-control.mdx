# SegmentedControl

An input-focused control for converting user intent into typed actions.


## What this is for
A horizontal row of toggle buttons where exactly one option is active. The active segment uses `ButtonVariant::Filled` with the theme's active color. Inactive segments use `ButtonVariant::Ghost`. The entire control is wrapped in a bordered, rounded container.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::SegmentedControl;

let node = SegmentedControl {
    ..Default::default()
}.into_node();
```

## Config surface
- `options`: `Vec<String>`
- `selected_index`: `usize`
- `on_change`: `Option<Arc<dyn Fn(usize) -> ActionEnvelope + Send + Sync>>`
