# Drawer

A navigation utility used to shape app-level flow and compositional patterns.


## What this is for
`Drawer` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::Drawer;

let node = Drawer {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `WidgetNodeId`
- `side`: `DrawerSide`
- `is_open`: `bool`
- `on_dismiss`: `Option<ActionEnvelope>`
- `content`: `Box<Node>`
- `width`: `Option<f32>`
