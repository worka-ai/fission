# RichText

A public widget/API entrypoint in the Fission surface.


## What this is for
`RichText` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::RichText;

let node = RichText {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<NodeId>`
- `runs`: `Vec<RichTextRun>`
- `inline_widgets`: `Vec<InlineWidgetSpan>`
- `annotations`: `Vec<IrRichTextAnnotation>`
- `semantics`: `Option<Semantics>`
- `width`: `Option<f32>`
- `height`: `Option<f32>`
- `min_width`: `Option<f32>`
- `max_width`: `Option<f32>`
- `min_height`: `Option<f32>`
- `max_height`: `Option<f32>`
- `wrap`: `bool`
