# HStack

A layout and composition primitive for predictable structure and spacing.


## What this is for
`HStack` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::HStack;

let node = HStack {
    ..Default::default()
}.into_node();
```

## Config surface
- `children`: `Vec<Node>`
- `spacing`: `Option<f32>`
