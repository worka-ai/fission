# RichTextRun

A public widget/API entrypoint in the Fission surface.


## What this is for
`RichTextRun` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::RichTextRun;

let node = RichTextRun {
    ..Default::default()
}.into_node();
```

## Config surface
- `text`: `String`
- `style`: `TextRunStyle`
- `semantics_label`: `Option<String>`
- `semantics_identifier`: `Option<String>`
- `spell_out`: `Option<bool>`
