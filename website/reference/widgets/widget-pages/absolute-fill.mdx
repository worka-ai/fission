# absolute_fill

A low-level composition utility for precise layout and rendering placement.


## What this is for
Wraps a child node in an `AbsoluteFill` layout node, causing it to stretch to fill its parent's bounds.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::absolute_fill;
let node = absolute_fill(fission_core::ui::Text::new("overlay").into_node());
```
