# ModalAction

A navigation utility used to shape app-level flow and compositional patterns.


## What this is for
`ModalAction` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::ModalAction;

let node = ModalAction {
    ..Default::default()
}.into_node();
```

## Config surface
- `label`: `String`
- `on_press`: `Option<ActionEnvelope>`
- `is_primary`: `bool`
