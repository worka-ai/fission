# Modal

A navigation utility used to shape app-level flow and compositional patterns.


## What this is for
`Modal` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::Modal;

let node = Modal {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `WidgetNodeId`
- `title`: `String`
- `content`: `Box<Node>`
- `is_open`: `bool`
- `on_dismiss`: `Option<ActionEnvelope>`
- `actions`: `Vec<ModalAction>`
- `width`: `Option<f32>`
