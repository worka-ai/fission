# GridItem

A layout and composition primitive for predictable structure and spacing.


## What this is for
`GridItem` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::GridItem;

let node = GridItem {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<NodeId>`
- `child`: `Box<Node>`
- `row_start`: `GridPlacement`
- `row_end`: `GridPlacement`
- `col_start`: `GridPlacement`
- `col_end`: `GridPlacement`
