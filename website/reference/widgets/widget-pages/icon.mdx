# Icon

A media or content surface used for visuals, media, and code presentation.


## What this is for
`Icon` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::Icon;

let node = Icon {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<NodeId>`
- `source`: `IconSource`
- `color`: `Option<Color>`
- `size`: `Option<f32>`
- `stroke`: `Option<Stroke>`
