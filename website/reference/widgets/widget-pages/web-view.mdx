# WebView

A media or content surface used for visuals, media, and code presentation.


## What this is for
`WebView` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_widgets::WebView;

let node = WebView {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `WidgetNodeId`
- `url`: `String`
- `user_agent`: `Option<String>`
