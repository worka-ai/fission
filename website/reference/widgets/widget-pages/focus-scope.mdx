# FocusScope

A public widget/API entrypoint in the Fission surface.


## What this is for
`FocusScope` is part of Fission's widget API and fits the deterministic, state-driven render model.

## Why you use it
- Keep UI behavior deterministic by composing from explicit nodes.
- Use it where behavior needs to stay testable and reviewable.
- Connect it through your existing action/reducer and environment flow.

## Minimal usage
```rust
use fission_core::ui::FocusScope;

let node = FocusScope {
    ..Default::default()
}.into_node();
```

## Config surface
- `id`: `Option<NodeId>`
- `children`: `Vec<Node>`
- `is_barrier`: `bool`
