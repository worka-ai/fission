# Layout widgets

Layout widgets define how children are arranged and sized.

Use container widgets when you need predictable structure:

- `Align`, `Column`, `Row`, `Grid`, `GridItem`, `LazyColumn`, `Overlay`, `SafeArea`, `Scroll`,
  `Spacer`, `ZStack`, `SplitView`, `SplitDirection`, `SimpleGrid`, `Wrap`, `Center`,
  `Positioned`, `Transform`, `Container`, and `GestureDetector`.

Use authoring helpers (`HStack`, `VStack`) when your team prefers semantic names for row/column
compositions.

A stable layout strategy keeps wide screens and narrow screens from feeling like different products.
