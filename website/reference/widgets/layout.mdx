# Layout widgets

Layout primitives control composition and geometry.

## Common work

- container direction (row/column),
- spacing and alignment,
- bounds and density behavior,
- nested composition for reusable screen sections.

## Practical guidance

Prefer explicit layout values for spacing and alignment, especially for wide screens and varying viewport widths.
