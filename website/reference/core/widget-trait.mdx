# Widget trait

All renderables implement the widget contract.

## Contract signature

```rust
fn build(&self, ctx: &mut BuildCtx<S>, view: &View<S>) -> Node
```

where `S` is the app state type.

## Responsibilities

- read-only access to current state via `view`,
- bind UI interactions through `ctx`,
- return a `Node` representing the next render tree.

## Rules of use

Keep build logic deterministic and side-effect free. If build does not stay predictable, diagnostics and replay become difficult.
