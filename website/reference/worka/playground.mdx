# Worka playground

The playground combines editor input and Worka execution with a fast runtime feedback loop.

- instant compile preview,
- logs and diagnostics available to users,
- reproducible snippet sharing.

## Workflow

1. edit sample,
2. run through Worka runtime,
3. validate UI and async behavior,
4. move stable work into the local codebase.
