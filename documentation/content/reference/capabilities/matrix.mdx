# Capability matrix reference

This reference uses the same terms as the guide matrix. **Yes** means the platform has a normal production path. **Provider** means the shared Fission API exists but the app or shell must supply a concrete provider and packaging metadata for that target. **Limited** means support depends heavily on browser, hardware, policy, distribution, or optional platform services. **No general API** means the platform does not expose a normal app-level API for the feature.
