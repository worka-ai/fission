# Dataset and dynamic

The Dataset and dynamic family contains 16 renderer-backed chart variants. Use this page as the entry point for choosing the right variant, then open the variant page for the screenshot, data shape, Rust API notes, and example code.

The chart family is still ordinary Fission UI. Each variant is constructed from typed Rust data, participates in layout and theme decisions, and can be driven by app state, reducers, resources, jobs, or services just like any other widget.

![Dataset stacked area screenshot](/img/charts/dataset-stack-area.png)

## Variants

| Chart | Data shape | Use when |
| --- | --- | --- |
| [Dataset stacked area](./dataset-stack-area) | Several named measures sharing ordered categories. | Use it when the data pipeline should stay visible instead of hidden in chart-specific arrays. |
| [Dynamic alert scatter](./dynamic-effect-alerts) | Small list of highlighted x/y samples. | Use it when outliers or active alerts need immediate attention. |
| [Dynamic brush scatter](./dynamic-brush-scatter) | Point samples plus a brush rectangle. | Use it when users need to select a region before drilling into data. |
| [Dynamic pictorial units](./dynamic-pictorial-units) | Category labels plus numeric values and a symbol choice. | Use it for unit counts that benefit from a branded or icon-like visual form. |
| [Dynamic polar score](./dynamic-polar-score) | Label/value pairs mapped around a circle. | Use it for cyclic or radial score comparisons. |
| [Dynamic radar health](./dynamic-radar-health) | Several same-length vectors across named indicators. | Use it for multidimensional status summaries. |
| [Dynamic sales funnel](./dynamic-funnel-sales) | Stage/value pairs ordered by process position. | Use it when the loss between stages matters more than exact axis measurement. |
| [Dynamic score gauge](./dynamic-gauge-score) | Single label/value pair scaled into a gauge arc. | Use it for scorecards where the main result should be immediately visible. |
| [Dynamic single-axis events](./dynamic-single-axis-events) | Position/value pairs on one continuous axis. | Use it for timelines or compact event strips. |
| [Dynamic speed gauge](./dynamic-gauge-speed) | Single label/value pair in a known range. | Use it for operational status values where one number dominates the view. |
| [Dynamic toolbox line](./dynamic-toolbox-line) | Ordered samples plus toolbox action configuration. | Use it when zoom, brush, restore, and export controls belong with the chart. |
| [Encoded bars dataset](./dataset-encoded-bars) | Records with category, group, and numeric value fields. | Use it when chart code should name fields once and reuse them across bar variants. |
| [Encoded lines dataset](./dataset-encoded-lines) | Records with ordered label, measure, and series fields. | Use it when the same dataset powers several line views. |
| [Visual map calendar](./visual-map-calendar) | Date/value pairs over a calendar range. | Use it to show daily volume, incidents, or activity in a compact year-like shape. |
| [Visual map heatmap](./visual-map-heatmap) | Grid cell coordinates with numeric intensity. | Use it for dense comparisons where color communicates rank faster than labels. |
| [Visual map scatter](./visual-map-scatter) | Triples of x, y, and magnitude values. | Use it when density and magnitude both matter. |

## Choosing within this family

Start with the user's question, not the visual effect. Pick the simplest variant that makes the answer clear, add interaction only when it reduces work for the user, and keep animation purposeful enough that it explains a change rather than decorating the page.
